# Vexe

Vexe is a tiny, yet efficient systems programming language.

Vexe is currently in an early development stage.

## Sub-projects

- [vexe](vexe) The Vexe Compiler frontend
- [vexe-llvm](vexe-llvm) Modern Kotlin bindings to the LLVM
